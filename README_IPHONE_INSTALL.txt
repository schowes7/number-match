# Number Match iPhone app setup

This project is now set up as an installable web app/PWA. Upload these files to a HTTPS host such as GitHub Pages, open the live link in Safari on your iPhone, then use Share → Add to Home Screen.

Files needed:
- index.html
- styles.css
- script.js
- manifest.webmanifest
- service-worker.js
- apple-touch-icon.png
- icon-192.png
- icon-512.png
